# 📖 About

The **ASME-AITR Student Chapter Website** is the official web presence of the **American Society of Mechanical Engineers (ASME)** Student Chapter at **Acropolis Institute of Technology and Research**.

It has been designed with a modern UI, responsive layout, animated interactions, and a clean engineering-inspired theme to showcase the chapter's activities, leadership, and achievements.
